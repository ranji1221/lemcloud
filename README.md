DeepOcean Cloud PlatForm
---------------------------------------

#### 项目简介
* Restful风格的JavaEE安全、稳定、快速的云平台
* 核心平台：
* (1) 基于OAuth2的认证授权平台 
* (2) 基于Jersey的webservice资源平台
---------------------------------------
#### 核心技术
* Spring Boot1.5.8+MyBatis3.2+Druid1.1.4+Apache Shiro 1.2
* SpringMVC+OAuth2+Jersey1.19.4+Shiro1.2
---------------------------------------
#### 项目管理与版本控制
* Maven3项目管理
* Git版本控制
